CSS Positioning - A Cheetsheet

A brief reference for CSS positioning properties

CSS Positioning
Attribute Values Note

position

static, relative, absolute, sticky

static does this.

relative does this.

absolute does this.

sticky does this.

top

Pixels, eg 10px, EM's, eg 1.5em, percentage, eg 10%

top sets space between the top of the element's container and the element.

For example, a declaration of top: 10px; would set the element 10 pixels from the top of it's containing element.
